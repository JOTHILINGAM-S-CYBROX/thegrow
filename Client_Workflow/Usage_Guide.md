# Usage Guide

## Admin Dashboard
1. **Login**: Use admin credentials to log in.
2. **Manage Bookings**: Navigate to the bookings section to view and manage bookings.
3. **Manage Menu**: Navigate to the menu section to add, edit, or delete items.

## Kitchen Dashboard
1. **Login**: Use kitchen credentials to log in.
2. **Manage Orders**: Navigate to the orders section to view and manage orders.

## Public Features
1. **Interactive Menu**: Customers can browse the menu on the homepage.
2. **Reservations**: Customers can book a table via the reservations page.
3. **Memberships**: Customers can sign up for memberships via the memberships page.

This document explains how to use the application.
