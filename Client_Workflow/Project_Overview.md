# Project Overview

## The Grove Application
The Grove is a Next.js-based web application designed to streamline operations for a venue. It includes dashboards for administrators and kitchen staff, as well as public-facing features for customers.

### Key Features:
- **Admin Dashboard**: Manage bookings, menu, and customer data.
- **Kitchen Dashboard**: Handle order management efficiently.
- **Public Features**: Interactive menu, reservations, and memberships.
- **API Integration**: RESTful APIs for seamless data handling.

### Technologies Used:
- **Next.js**: Framework for server-side rendering and routing.
- **React**: Component-based architecture.
- **Tailwind CSS**: For styling.
- **Node.js**: Backend runtime.
- **MongoDB**: Database for storing application data.
