# Key Workflows

## Admin Dashboard
- **Login**: Admins log in to access the dashboard.
- **Bookings Management**: View and manage venue bookings.
- **Menu Management**: Add, edit, or delete menu items.

## Kitchen Dashboard
- **Login**: Kitchen staff log in to access the dashboard.
- **Order Management**: View and manage customer orders.

## Public Features
- **Interactive Menu**: Customers can browse the menu.
- **Reservations**: Customers can book a table.
- **Memberships**: Customers can sign up for memberships.

## API Workflows
- **Authentication**: OTP-based login for customers.
- **Bookings API**: Endpoints for creating and managing bookings.
- **Orders API**: Endpoints for creating and managing orders.

This document explains the main workflows of the application.
